# C & C++ Computational Algorithms and Data Structures

A comprehensive collection of programming implementations focusing on computational mathematics, dynamic memory management, data structures, and Object-Oriented Programming (OOP).

## Repository Structure

- **Fundamental_Algorithms:** Mathematical function evaluations, recursive sequence solving, Fourier series approximations, and custom text manipulation (Caesar Cipher).
- **Arrays_and_Matrices:** Multi-dimensional array handling, vector computations, and mathematical properties validation.
- **Memory_and_File_Handling:** Advanced pointer manipulation, dynamic memory allocation (`malloc`/`free`), and binary/text file I/O operations.
- **Advanced_Structures:** Implementation of custom Data Structures in C (Singly Linked Lists) and Object-Oriented Programming in C++ (Matrix abstraction with Operator Overloading).

## How to Run
- For C scripts: Compile using `gcc filename.c -o program -lm` and run `./program`
- For C++ scripts: Compile using `g++ filename.cpp -o program` and run `./program`
